# JS Assignment

## Approach

Q1: Loop from L to R, check divisibility, digit sum prime, and no zero.

Q2: Apply transformation 3 times, check range and middle digit.

Q3: Try all X from 0 to 100000, check palindrome + divisibility.

Q4: Follow step-by-step fare rules and rounding logic.

Q5: Keep adding numbers (skip divisible ones) until sum ≥ N. 

Q6: Compute score and apply rules.

## Complexity

Q1: O(N * digits)
Q2: O(1)
Q3: O(100000 * digits)
Q4: O(1)
Q5: O(m)
Q6: O(1)
